üß† Development Approach (Vibe Coding / Command Prompts)

This project is intentionally built using a prompt-first engineering methodology, where:

Curriculum logic is defined before UI

CAPS structure drives data models

Lessons, activities, and assessments are generated from explicit prompts

The AI agent acts as a coding assistant, not a decision-maker

All outputs are deterministic, not probabilistic

This ensures:

Predictable learning outcomes

Audit-friendly logic (important for DoE & schools)

Faster prototyping without sacrificing structure

Think of this as software architecture written in English first, then compiled into code.
